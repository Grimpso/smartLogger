WHAT IS OOP?
oop is way of writing programs by thinking in terms of real-world objects insted of functions and variables
it improves ----scalability,maintainability,reusablity,security etc;

WHAT IS CLASS?
class is a blueprint which defines what will object have(data) and what will object do(functions);
example->blueprint of house

WHAT IS AN OBJECT?
object is a real instance which is created from a class
ex->class-car
    object->my car
